# Day 31: 机器翻译项目实战

本文件夹包含机器翻译项目的实战内容，基于 Transformer 模型。

## 学习内容

- **transformer_带bleu-aliyun.ipynb**: 阿里云训练版本
- **transformer_带bleu-autodl训练结果.ipynb**: AutodL 训练结果
- **生成器.ipynb**: Python 生成器
- **指数对数变换.ipynb**: 数据变换方法
- **data_multi30k.py**: 数据处理脚本
- **data_multi30k.sh**: 数据处理脚本
- **wmt16 back/**: 原始数据
- **wmt16_cut/**: 处理后数据

## 核心知识点

- 完整机器翻译项目
- 模型训练与评估
- BLEU 分数计算
- 推理部署

## 适合人群

NLP 学习者，需要掌握机器翻译项目

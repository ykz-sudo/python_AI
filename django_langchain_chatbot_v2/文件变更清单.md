## API变更

### 请求格式变化

**原格式**:
```json
{
  "message": "用户消息",
  "user": "anonymous"
}
```

**新格式**:
```json
{
  "message": "用户消息",
  "user": "anonymous",
  "session_id": "uuid-string"  // 可选，新增
}
```

### 响应格式变化

**原格式**:
```json
{
  "answer": "AI回复",
  "sources": ["local_faiss"]
}
```

**新格式**:
```json
{
  "answer": "AI回复",
  "sources": ["local_faiss"],
  "session_id": "uuid-string"  // 新增
}
```

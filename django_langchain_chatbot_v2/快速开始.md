# 快速开始指南

## 前置要求

- Python 3.8+
- OpenAI API Key
- 基本的命令行操作知识

## 5分钟快速部署

### 步骤1: 克隆或下载项目

```bash
cd django_langchain_chatbot_v2
```

### 步骤2: 安装依赖

```bash
pip install -r requirements.txt
```

如果遇到安装问题，可以尝试：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**验证安装**（推荐）:
```bash
python check_versions.py
```

这个脚本会检查所有依赖是否正确安装。

### 步骤3: 配置OpenAI API Key

创建 `.env` 文件（或复制 `.env.template`）：

```bash
# Windows
copy .env.template .env

# Linux/Mac
cp .env.template .env
```

编辑 `.env` 文件，添加你的API Key：
```
OPENAI_API_KEY=sk-your-api-key-here
OPENAI_API_BASE=https://api.openai.com/v1  # 可选，使用代理时修改
```

### 步骤4: 初始化数据库

```bash
python manage.py migrate
```

预期输出：
```
Operations to perform:
  Apply all migrations: chatbot_app
Running migrations:
  Applying chatbot_app.0001_initial... OK
  Applying chatbot_app.0002_chatsession_chatmessage... OK
```

### 步骤5: 准备知识库（可选）

如果你有自己的文档，放到 `data/docs/` 目录：

```bash
# 将你的txt文件放到这里
data/docs/your_document.txt
```

然后构建索引：
```bash
python manage.py build_faiss_index
```

如果没有文档，系统会使用默认的 `data/docs/all_sheets.txt`。

### 步骤6: 启动服务

```bash
python manage.py runserver
```

看到以下输出表示成功：
```
Starting development server at http://127.0.0.1:8000/
Quit the server with CTRL-BREAK.
```

### 步骤7: 开始使用

打开浏览器访问：
```
http://localhost:8000/
```

## 常见问题

### Q1: 提示 "知识库未构建"

**解决方案**:
```bash
python manage.py build_faiss_index
```

### Q2: OpenAI API 连接失败

**可能原因**:
1. API Key 错误
2. 网络问题
3. API额度用完

**解决方案**:
- 检查 `.env` 文件中的 API Key
- 如果在国内，可能需要配置代理
- 检查 OpenAI 账户余额

### Q3: 导入错误 "No module named 'langchain'"

**解决方案**:
```bash
pip install -r requirements.txt
```

### Q4: 数据库迁移失败

**解决方案**:
```bash
# 删除数据库（会丢失数据）
rm db.sqlite3

# 重新迁移
python manage.py migrate
```

### Q5: 端口8000已被占用

**解决方案**:
```bash
# 使用其他端口
python manage.py runserver 8001
```

## 进阶配置

### 使用其他数据库

编辑 `chatbot_project/settings.py`:

```python
# PostgreSQL 示例
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'chatbot_db',
        'USER': 'your_username',
        'PASSWORD': 'your_password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```

### 修改AI模型

编辑 `chatbot_app/rag.py` 和 `chatbot_app/views.py`:

```python
# 将 gpt-5-nano 改为其他模型
llm = ChatOpenAI(model="gpt-4", temperature=0.7)
```

### 调整对话历史长度

编辑 `chatbot_app/views.py`:

```python
# 修改这一行，调整保留的历史消息数量
history_messages = ChatMessage.objects.filter(session=session).order_by('created_at')[:-1]
```

编辑 `chatbot_app/rag.py`:

```python
# 修改这一行，调整传递给AI的历史消息数量
recent_history = chat_history[-10:]  # 改为你想要的数量
```

### 定期清理旧会话

创建定时任务（Linux crontab 示例）:

```bash
# 每天凌晨2点清理30天前的会话
0 2 * * * cd /path/to/project && python manage.py cleanup_old_sessions --days 30
```

## 生产部署建议

### 1. 使用生产级服务器

```bash
# 安装 gunicorn
pip install gunicorn

# 启动
gunicorn chatbot_project.wsgi:application --bind 0.0.0.0:8000
```

### 2. 配置Nginx反向代理

```nginx
server {
    listen 80;
    server_name your-domain.com;

    location / {
        proxy_pass http://127.0.0.1:8000;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    location /static/ {
        alias /path/to/project/static/;
    }
}
```

### 3. 使用环境变量

不要在代码中硬编码敏感信息，使用环境变量：

```python
import os
OPENAI_API_KEY = os.getenv('OPENAI_API_KEY')
```

### 4. 启用HTTPS

使用 Let's Encrypt 免费SSL证书：

```bash
sudo certbot --nginx -d your-domain.com
```

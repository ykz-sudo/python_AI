========================================
Django LangChain 聊天机器人 v2.0
多轮对话版本 - 快速开始指南
========================================

✅ 已完成版本适配！
   LangChain 0.3.7 完全兼容

========================================
第一步：验证环境
========================================

运行版本检查脚本：
> python check_versions.py

如果所有检查都通过（显示绿色 ✅），继续下一步。
如果有错误，请先安装依赖：
> pip install -r requirements.txt

========================================
第二步：配置 API Key
========================================

1. 确保 .env 文件存在并包含：
   OPENAI_API_KEY=你的API密钥

2. 如果没有 .env 文件，创建它：
   复制 .env.template 为 .env
   然后编辑添加你的 API Key

========================================
第三步：初始化数据库
========================================

运行数据库迁移：
> python manage.py migrate

预期看到：
  Applying chatbot_app.0001_initial... OK
  Applying chatbot_app.0002_chatsession_chatmessage... OK

========================================
第四步：构建知识库（可选）
========================================

如果你有自己的文档：
1. 将 .txt 文件放到 data/docs/ 目录
2. 运行：python manage.py build_faiss_index

如果使用默认文档，直接运行：
> python manage.py build_faiss_index

========================================
第五步：启动服务
========================================

启动 Django 服务器：
> python manage.py runserver

看到以下信息表示成功：
  Starting development server at http://127.0.0.1:8000/

========================================
第六步：开始使用
========================================

1. 打开浏览器访问：
   http://localhost:8000/

2. 开始对话！系统会自动记住上下文

3. 查看对话历史：
   http://localhost:8000/sessions/

========================================
功能测试
========================================

运行自动化测试脚本：
> python test_multiround_chat.py

这会测试：
- 多轮对话功能
- 上下文理解
- 联系方式提取
- 会话管理

========================================
主要功能
========================================

✅ 多轮对话 - AI 记住之前的对话内容
✅ 智能问答 - 基于知识库的 RAG 系统
✅ 联系方式提取 - 自动识别手机、微信、QQ
✅ 负面反馈检测 - 自动记录用户不满
✅ 会话历史 - 所有对话持久化保存
✅ 清空对话 - 随时开始新会话

========================================
常用命令
========================================

# 启动服务器
python manage.py runserver

# 构建索引
python manage.py build_faiss_index

# 清理旧会话（30天前）
python manage.py cleanup_old_sessions --days 30

# 运行测试
python test_multiround_chat.py

# 检查版本
python check_versions.py

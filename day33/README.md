# Day 33: ChatGLM 与 PEFT 微调

本文件夹包含 ChatGLM 大语言模型和 PEFT 高效微调技术的内容。

## 学习内容

- **chatglm3_6b官方模型预测_在线下载_py11.ipynb**: ChatGLM3-6B 模型使用
- **ChatGLM官方模型预测-看看即可.ipynb**: ChatGLM 预测示例
- **PEFT（Parameter_Efficient_Fine_Tuning_讲解).ipynb**: PEFT 微调详解
- **模型结构.ipynb**: 模型结构分析
- **相对位置编码与旋转位置编码.ipynb**: 位置编码
- **glm_4v_9b (3).ipynb**: GLM4-Vision 9B

## 核心知识点

- ChatGLM 模型架构
- PEFT 高效微调
- LoRA 微调
- 位置编码：相对位置编码、旋转位置编码
- 大模型部署

## 适合人群

大语言模型学习者，需要掌握 LLM 微调技术

```bash
pip install transformers peft torch accelerate
```

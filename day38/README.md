# Day 38: Dify 平台与 RAG 系统搭建

本文件夹包含 Dify 平台和 RAG 系统搭建的内容。

## 学习内容

- **dify/**: Dify 智能体平台
  - **智能体.ipynb**: Dify 智能体使用
  - **dify_test.py**: Dify API 测试
  - **一键生成行业调研报告.yml**: 工作流配置

- **rag/**: RAG 系统
  - **deepseek-rag.ipynb**: RAG 实战
  - **davinci.txt**: 测试数据
  - **your_env/**: 环境配置
  - **.env**: 环境变量

## 核心知识点

- Dify 智能体平台
- Dify 工作流
- RAG 系统搭建
- 向量数据库
- API 集成

大语言模型应用开发者，需要掌握 LLM 应用部署

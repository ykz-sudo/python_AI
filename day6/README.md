# Day 6: 面向对象进阶

## 学习内容

- **1-对象方法中使用super.py**: super 在对象方法中的使用
- **2-子类中是否可以使用父类的私有属性.py**: 私有属性访问
- **3-多态.py**: 多态特性
- **4-类属性.py**: 类属性
- **5-类方法与静态方法.py**: 类方法和静态方法
- **6-单例模式.py**: 单例模式实现
- **7-练习身份运算符.py**: 身份运算符练习
- **8-捕获不同类型异常.py**: 异常捕获
- **9-得到异常发生在哪一行.py**: 异常定位
- **10-异常发生在具体的模块-函数-行.py**: 异常追踪
- **11-异常的传递.py**: 异常传递机制
- **12-抛出异常.py**: raise 抛出异常
- **13-断言异常.py**: 断言语句
- **14-自定义类型异常.py**: 自定义异常类
- **15-使用模块.py**: 模块导入与使用
- **16-理解模块属性.py**: 模块属性
- **17-使用包.py**: 包的使用
- **18-使用包2.py**: 包进阶
- **19-使用第三方的包或者模块.py**: 第三方库使用
- **20-文件操作.py**: 文件操作基础
- **exercise.py**: 练习题

- 多态：不同对象对同一消息的不同响应
- 类属性 vs 实例属性
- 类方法 vs 静态方法
- 单例模式
- 异常处理：try-except-finally
- 自定义异常
- 模块与包
- Python 标准库

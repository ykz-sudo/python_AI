# Day 9: NumPy 与 Matplotlib 入门

本文件夹包含 NumPy 数值计算和 Matplotlib 数据可视化的入门内容。

## 学习内容

- **1-正则查找第二个.py**: 正则表达式练习
- **2-sorted的使用.py**: sorted 函数使用
- **3-jupyter使用讲解.ipynb**: Jupyter Notebook 使用指南
- **4-matplotlib.ipynb**: Matplotlib 数据可视化
- **5-numpy.ipynb**: NumPy 基础
- **format_vs_fstring_demo.py**: 字符串格式化对比

## 核心知识点

- NumPy：数组创建、索引、切片、运算
- Matplotlib：折线图、柱状图、散点图等
- Jupyter Notebook：交互式编程环境
- 数据可视化基础

## 适合人群

Python 初学者，需要掌握数据分析和可视化基础

## 环境依赖

```bash
pip install numpy matplotlib jupyter
```

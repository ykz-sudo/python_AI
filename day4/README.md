# Day 4: 函数基础

本文件夹包含 Python 函数式编程的相关内容。

## 学习内容

- **1-函数基础.py**: 函数定义与调用
- **2-函数嵌套.py**: 函数的嵌套调用
- **3-局部变量与全局变量.py**: 变量作用域
- **4-把函数做为参数传递.py**: 高阶函数
- **5-函数进阶.py**: 函数进阶特性
- **6-不定长参数.py**: *args 和 **kwargs
- **7-拆包.py**: 字典和列表的拆包
- **8-引用.py**: 引用与可变对象
- **9-匿名函数.py**: lambda 表达式
- **10-递归.py**: 递归函数
- **11-Python的内存管理.py**: 内存管理机制
- **12-类与对象.py**: 类与对象基础
- **13-内置方法.py**: Python 内置方法
- **exercise.py**: 练习题

## 核心知识点

- 函数定义与调用
- 参数传递：位置参数、关键字参数、默认参数
- 不定长参数：*args, **kwargs
- 变量作用域：局部变量、全局变量
- 匿名函数：lambda 表达式
- 递归函数
- 引用与可变对象

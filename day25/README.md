# Day 25: LSTM 文本生成

本文件夹包含 LSTM 网络的文本生成实战内容。

## 学习内容

- **chapter_6/**:
  - **01_embedding_padding_pooling.ipynb**: 词向量基础
  - **02_embedding_rnn.ipynb**: RNN 基础
  - **03_text_generation.ipynb**: 文本生成
  - **03_text_generation_aliyun.ipynb**: 阿里云版本
  - **06_embedding_lstm_subword.ipynb**: Subword 词表
- **shakespeare.txt**: 莎士比亚文本数据集
- **imdb_bpe_code/**: IMDB BPE 编码

## 核心知识点

- LSTM 长短期记忆网络
- 文本生成
- 字符级/词级模型
- BPE 分词
- 序列到序列

NLP 学习者，需要掌握文本生成技术

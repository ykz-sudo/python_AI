# Day 19: 深度神经网络进阶

本文件夹包含深度神经网络进阶内容，包括 Dropout、BatchNorm、SeLU 等技术。

## 学习内容

- **chapter2/**:
  - **01-classification_model.ipynb**: 分类模型基础
  - **02-classification_model_more_control.ipynb**: 进阶控制
  - **03_regression.ipynb**: 回归模型
  - **04_classification_model-dnn.ipynb**: DNN 分类
  - **05_classification_model_dnn_bn.ipynb**: BatchNorm
  - **06_classification_model-dnn_selu.ipynb**: SELU 激活函数
  - **07_classification_model-dnn_dropout.ipynb**: Dropout
  - **wangdao_train.py**: 训练代码

## 核心知识点

- Dropout 正则化
- BatchNorm 批归一化
- SELU 激活函数
- 权重初始化
- 学习率调度

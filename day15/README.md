# Day 15: 逻辑回归与聚类算法

本文件夹包含逻辑回归和聚类算法的相关内容。

## 学习内容

- **1-线性回归-Lasso回归-岭回归.ipynb**: 线性回归进阶
  - 岭回归 Ridge
  - Lasso 回归
  - 正则化
- **2-逻辑回归.ipynb**: 逻辑回归
  - 二分类问题
  - 乳腺癌预测
- **3-聚类.ipynb**: 聚类算法
  - K-Means 聚类
  - DBSCAN
- **data/**:
  - breast-cancer-wisconsin.csv: 乳腺癌数据集
  - customers.csv: 客户数据集

## 核心知识点

- 逻辑回归：sigmoid 函数、交叉熵损失
- 正则化：L1（Lasso）、L2（Ridge）
- 分类评估指标：准确率、召回率、F1
- 聚类算法：K-Means、DBSCAN
- 轮廓系数

## 适合人群

机器学习初学者，需要掌握分类和聚类算法

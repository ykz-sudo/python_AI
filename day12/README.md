# Day 12: Scikit-learn 数据预处理与特征工程

本文件夹包含 Scikit-learn 数据预处理和特征工程的相关内容。

## 学习内容

- **1-sklearn-case1.ipynb**: Scikit-learn 入门案例
- **2-加载sklearn的各种自带数据集.ipynb**: sklearn 内置数据集
- **3-特征提取.py**: 特征提取方法
- **4-数值特征预处理.ipynb**: 数值特征预处理
  - 标准化、归一化
  - 缺失值处理
- **5-特征选择.ipynb**: 特征选择方法
- **california_housing.csv**: 加州房价数据集

## 核心知识点

- Scikit-learn 常用数据集
- 特征提取：CountVectorizer、TfidfVectorizer
- 数值预处理：StandardScaler、MinMaxScaler
- 缺失值处理：SimpleImputer
- 特征选择：SelectKBest、PCA

## 适合人群

机器学习初学者，需要掌握数据预处理技能

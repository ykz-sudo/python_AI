# Day 5: 面向对象编程基础

本文件夹包含 Python 面向对象编程的基础内容。

## 学习内容

- **1-摆放家具.py**: 面向对象案例
- **2-封装案例1.py**: 封装案例
- **3-封装案例2.py**: 封装进阶案例
- **5-私有方法.py**: 私有属性与方法
- **6-继承.py**: 继承基础
- **7-继承初始化带参数.py**: 子类初始化
- **8-多继承.py**: 多继承
- **9-多继承的方法查找问题.py**: MRO 方法解析顺序
- **10-子类调用父类中的方法.py**: super() 函数
- **exercise.py**: 练习题

## 核心知识点

- 类与对象的概念
- 封装：属性和方法的私有化
- 继承：单继承、多继承
- super() 函数：调用父类方法
- MRO：方法解析顺序

# Day 10: NumPy 进阶与数据文件处理

本文件夹包含 NumPy 进阶内容和数据文件处理的实战内容。

## 学习内容

- **1-numpy.ipynb**: NumPy 基础操作
- **2-numpy-轴之后.ipynb**: NumPy 轴与维度操作
- **3-numpy读取文件.ipynb**: NumPy 文件读写
- **youtube_video_data/**: YouTube 视频数据
  - USvideos.csv: 美国视频数据
  - GBvideos.csv: 英国视频数据
  - US_video_data_numbers.csv: 美国数值数据
  - GB_video_data_numbers.csv: 英国数值数据

## 核心知识点

- NumPy 数组的多维操作
- 轴（axis）的概念
- NumPy 文件读写：save、load、savetxt、loadtxt
- CSV 数据处理
- 数据清洗基础

## 适合人群

Python 初学者，需要掌握数据处理技能

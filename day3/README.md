# Day 3: 数据结构

本文件夹包含 Python 核心数据结构的相关内容。

## 学习内容

- **1-选择循环题目练习.py**: 选择与循环语句练习
- **2-字符串.py**: 字符串操作
- **3-列表.py**: 列表操作
- **4-元组.py**: 元组操作
- **5-字典.py**: 字典操作
- **6-集合.py**: 集合操作
- **9-推导式.py**: 列表/字典/集合推导式
- **exercise.py**: 练习题

## 核心知识点

- 字符串：切片、格式化、常用方法
- 列表：增删改查、切片操作
- 元组：不可变序列
- 字典：键值对操作
- 集合：去重、集合运算
- 推导式：简洁创建列表/字典/集合

## 适合人群

Python 初学者，需要掌握 Python 数据结构

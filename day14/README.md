# Day 14: 朴素贝叶斯与线性回归

本文件夹包含朴素贝叶斯分类和线性回归的相关内容。

## 学习内容

- **1-朴素贝叶斯实现新闻分类.ipynb**: 朴素贝叶斯新闻分类
  - 文本特征提取
  - 多分类任务
- **3-线性回归.ipynb**: 线性回归
  - 房价预测案例
- **data/titanic.csv**: 泰坦尼克号数据集

## 核心知识点

- 朴素贝叶斯分类器：MultinomialNB、BernoulliNB、GaussianNB
- 文本分类：TF-IDF
- 线性回归：最小二乘法
- 回归评估指标：MSE、RMSE、R²

## 适合人群

机器学习初学者，需要掌握分类和回归算法

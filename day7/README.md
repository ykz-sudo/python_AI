# Day 7: 文件操作与正则表达式

本文件夹包含 Python 文件操作和正则表达式的相关内容。

## 学习内容

- **1-文件操作.py**: 文件打开、读取、写入
- **2-文件操作函数.py**: 文件操作函数
- **3-使用with.py**: with 上下文管理器
- **4-seek操作与练习.py**: 文件指针操作
- **5-文件及目录练习.py**: 目录操作
- **6-使用eval.py**: eval 函数使用
- **7-深copy与浅copy.py**: 深拷贝与浅拷贝
- **8-正则表达式.py**: 正则表达式基础
- **practice.py**: 练习题

## 核心知识点

- 文件操作：open、read、write、close
- with 上下文管理器
- 文件指针：seek、tell
- 目录操作：os、pathlib
- 深拷贝 vs 浅拷贝
- 正则表达式：match、search、findall、sub

## 适合人群

Python 初学者，需要掌握文件处理和文本解析技能

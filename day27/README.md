# Day 27: 注意力机制进阶与 BLEU 评估

本文件夹包含注意力机制进阶内容和 BLEU 评估的实战内容。

## 学习内容

- **project/**:
  - **seq2seq_attention_torch-2026.ipynb**: Seq2Seq 完整实现
  - **seq2seq_attention_torch-2026-autodl_lstm.ipynb**: LSTM 版本
  - **seq2seq_attention_torch-2026-autodl_gru.ipynb**: GRU 版本
  - **seq2seq_attention_torch-2026-autodl_4layers.ipynb**: 4层模型
  - **seq2seq_attention_torch-2026-autodl_4grams.ipynb**: 4-gram 模型
  - **data_multi30k.py**: 数据处理
- **计算bleu.ipynb**: BLEU 分数计算

## 核心知识点

- 注意力机制变体
- 多层 Seq2Seq
- GRU vs LSTM
- BLEU 评估
- 机器翻译优化

## 适合人群

NLP 学习者，需要深入理解注意力机制

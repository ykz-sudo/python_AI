# Day 29: Transformer 架构详解

本文件夹包含 Transformer 架构的详细内容和机器翻译实战。

## 学习内容

- **transformer_bleu-autodl copy.ipynb**: Transformer 完整实现
- **transformer_bleu-autodl_tune.ipynb**: 超参数调优
- **生成器.ipynb**: 生成器模式
- **data_multi30k.py**: 数据处理脚本
- **data_multi30k.sh**: 数据处理脚本
- **wmt16 back/**: 原始翻译数据
- **wmt16_cut/**: 处理后的翻译数据
- **README.md**: 项目说明
- **hyperparam_tuning.md**: 超参数调优笔记

- Transformer 架构

- Encoder-Decoder 结构
- BLEU 评估
